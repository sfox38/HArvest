HArvest

Your smart home, embedded anywhere.

  • Embed live, interactive Home Assistant cards in any web page or WordPress site - lightweight enough to run on old tablets, phones, and kiosks.
  • Every card uses a scoped, cryptographic token - visitors don't need a Home Assistant login.
  • Set up in minutes, no additional expertise required - install from HACS and build cards in a point-and-click panel.
  • Self-hosted - no extra server, no third-party relay, no cloud account.
  • Make it yours - build custom themes, use third-party themes, or reuse the dashboards you already have.
  • 100% Free. Open source, MIT licensed.
